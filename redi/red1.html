Introducción a Java

¿Qué es Java?

Java es un lenguaje de programación de propósito general, orientado a objetos y diseñado para tener un alto nivel de portabilidad. Fue creado por Sun Microsystems en 1995 y actualmente es desarrollado y mantenido por Oracle Corporation.

Características principales de Java

  • Orientado a objetos: Java sigue el paradigma de programación orientada a objetos, lo que significa que se organiza en clases y objetos, permitiendo la reutilización de código y la implementación de conceptos como la herencia y el polimorfismo.
  • Portabilidad: Los programas escritos en Java pueden ser ejecutados en cualquier plataforma que tenga una máquina virtual Java (JVM) instalada, lo que hace que los programas Java sean altamente portables e independientes del sistema operativo.
  • Seguridad: Java tiene un modelo de seguridad robusto que protege contra amenazas como el acceso no autorizado a recursos y la ejecución de código malicioso.
  • Simplicidad: Java fue diseñado para ser un lenguaje fácil de aprender y entender, con una sintaxis clara y concisa.

Ejemplo de código Java:

      
        public class HelloWorld {
          public static void main(String[] args) {
            System.out.println("¡Hola, mundo!");
          }
        }

El código anterior es un programa Java muy básico que muestra el mensaje "¡Hola, mundo!" en la consola. Es común que este sea el primer programa que se escribe al aprender Java.

Sintaxis básica de Java

Clase principal

Un programa en Java comienza con una clase principal que tiene el siguiente formato:

public class Main {
    public static void main(String[] args) {
      // Código a ejecutar
    }
  }

Declaración de variables

Para declarar una variable en Java, se utiliza la siguiente sintaxis:

tipoDeDato nombreDeVariable;

Por ejemplo:

int edad;

Comentarios

Los comentarios en Java se utilizan para documentar el código y no se ejecutan. Existen dos tipos de comentarios:

// Este es un comentario de una línea
  
  /* Este es un comentario
     de múltiples líneas */

Funciones

En Java, se pueden definir funciones utilizando la siguiente sintaxis:

tipoDeDato nombreDeFuncion(tipoDeDato parametro1, tipoDeDato parametro2) {
    // Código de la función
    return resultado;
  }
