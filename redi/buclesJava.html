Bucles en Java

Bucle For

El bucle for se utiliza para repetir un bloque de código un número específico de veces. Tiene tres partes: una inicialización, una condición y una actualización.

      
        for (int i = 0; i < 5; i++) {
          System.out.println("Iteración " + i);
        }

En el ejemplo anterior, el bucle for se ejecutará 5 veces. La variable i se inicializa en 0, y mientras i sea menor que 5, se ejecutará el bloque de código dentro del bucle. Después de cada iteración, se incrementa el valor de i en 1.

Bucle While

El bucle while se utiliza para repetir un bloque de código mientras se cumpla una condición.

      
        int contador = 0;

        while (contador < 3) {
          System.out.println("Iteración " + contador);
          contador++;
        }

En el ejemplo anterior, el bucle while se ejecutará mientras contador sea menor que 3. En cada iteración, se muestra el mensaje "Iteración" seguido del valor de contador y luego se incrementa contador en 1.

Bucle Do-While

El bucle do-while es similar al bucle while, pero la diferencia es que se ejecuta al menos una vez antes de verificar la condición.

      
        int x = 0;

        do {
          System.out.println("Iteración " + x);
          x++;
        } while (x < 3);

En el ejemplo anterior, el bucle do-while se ejecutará al menos una vez, ya que la condición se verifica después de cada iteración. Mientras x sea menor que 3, se mostrará el mensaje "Iteración" seguido del valor de x y luego se incrementará x en 1.
