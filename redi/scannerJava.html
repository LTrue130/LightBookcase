Uso del Scanner en Java

El Scanner es una clase en Java que se utiliza para obtener la entrada del usuario desde la consola o desde otro flujo de datos. Proporciona métodos para leer diferentes tipos de datos, como enteros, flotantes, cadenas, etc.

Ejemplo de uso del Scanner

El siguiente ejemplo muestra cómo usar el Scanner para leer un número entero ingresado por el usuario:

    
import java.util.Scanner;

public class ScannerExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Ingrese un número entero: ");
        int numero = scanner.nextInt();
        
        System.out.println("El número ingresado es: " + numero);
        
        scanner.close();
    }
}

En este ejemplo, primero se importa la clase Scanner del paquete java.util. Luego, se crea una instancia de Scanner pasando System.in como argumento, lo que indica que se leerá la entrada desde la consola.

Se muestra un mensaje al usuario para que ingrese un número entero. Luego, se utiliza el método nextInt() del objeto Scanner para leer el número entero ingresado por el usuario y se almacena en la variable numero.

Finalmente, se imprime el número ingresado en la consola. Es importante cerrar el objeto Scanner llamando al método close() para liberar los recursos utilizados.

Otros métodos útiles del Scanner

Además de nextInt(), el Scanner proporciona otros métodos para leer diferentes tipos de datos:

  • nextLine(): Lee una línea completa de texto ingresada por el usuario. Por ejemplo:
    •       
System.out.print("Ingrese su nombre: ");
String nombre = scanner.nextLine();
System.out.println("Hola, " + nombre);
  • nextDouble(): Lee un número en formato de punto flotante ingresado por el usuario. Por ejemplo:
    •       
System.out.print("Ingrese un número decimal: ");
double decimal = scanner.nextDouble();
System.out.println("El número ingresado es: " + decimal);
  • nextBoolean(): Lee un valor booleano (true o false) ingresado por el usuario. Por ejemplo:
    •       
System.out.print("¿Está de acuerdo? (true/false): ");
boolean acuerdo = scanner.nextBoolean();
if (acuerdo) {
    System.out.println("Está de acuerdo");
} else {
    System.out.println("No está de acuerdo");
}

Estos son solo algunos de los métodos disponibles en la clase Scanner de Java. Puedes consultar la documentación oficial de Java para obtener más información sobre los diferentes métodos y opciones de lectura proporcionados por el Scanner.

Cierre del Scanner

Es importante cerrar el objeto Scanner cuando ya no se necesite. El cierre del Scanner libera los recursos utilizados y evita posibles fugas de memoria. Se puede cerrar llamando al método close() del objeto Scanner. Por ejemplo:

    
scanner.close();

Es recomendable cerrar el Scanner al finalizar su uso o cuando ya no se vaya a utilizar más. Si no se cierra adecuadamente, puede provocar problemas en la aplicación.
