Bucles en Python

Bucle while

El bucle while se utiliza para repetir un bloque de código mientras se cumpla una condición. Aquí tienes un ejemplo:

      
        contador = 0
        while contador < 5:
            print('El contador es', contador)
            contador += 1

Bucle for

El bucle for se utiliza para recorrer una secuencia de elementos (como una lista o una cadena de texto). Aquí tienes un ejemplo:

      
        frutas = ['manzana', 'banana', 'cereza']
        for fruta in frutas:
            print(fruta)

Control de bucles

En Python, puedes controlar la ejecución de un bucle utilizando las siguientes declaraciones:

  • break: Termina el bucle y sale de él.
  • continue: Salta a la siguiente iteración del bucle sin ejecutar el código restante.

Aquí tienes un ejemplo que utiliza estas declaraciones:

      
        for numero in range(10):
            if numero == 7:
                break
            if numero == 3:
                continue
            print(numero)
