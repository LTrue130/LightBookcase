Introducción a Python

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Se destaca por su sintaxis clara y legible, lo que facilita su aprendizaje y comprensión.

Características de Python

  • Sintaxis simple y legible
  • Tipado dinámico
  • Soporte para programación orientada a objetos
  • Amplia biblioteca estándar
  • Interpretado, lo que significa que no requiere una etapa de compilación

Ejemplo de código

A continuación se muestra un ejemplo de código Python que imprime el mensaje "¡Hola, Python!" en la consola:

      
        print("¡Hola, Python!")

Sintaxis de Python

Comentarios

Los comentarios en Python se utilizan para agregar notas y explicaciones en el código. Se pueden crear comentarios de una sola línea utilizando el símbolo #. Por ejemplo:

          
            # Esto es un comentario de una sola línea

También se pueden crear comentarios de varias líneas utilizando tres comillas simples ''' o tres comillas dobles """. Por ejemplo:

          
            '''
            Este es un comentario
            de varias líneas
            '''

Variables

En Python, las variables se utilizan para almacenar datos. No es necesario declarar el tipo de variable antes de su uso, ya que Python es un lenguaje de tipado dinámico. Por ejemplo:

          
            nombre = 'Juan'
            edad = 25

Condicionales

Python utiliza las palabras clave if, elif y else para crear estructuras condicionales. Por ejemplo:

          
            if edad < 18:
                print('Eres menor de edad')
            elif edad >= 18 and edad < 65:
                print('Eres mayor de edad')
            else:
                print('Eres adulto mayor')

Bucles

Python ofrece diferentes bucles para repetir una serie de instrucciones. Los bucles más comunes son for y while. Por ejemplo:

          
            for i in range(5):
                print(i)
                
            while contador < 10:
                print(contador)
                contador += 1
