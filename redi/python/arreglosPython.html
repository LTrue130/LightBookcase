Arreglos en Python

Arreglos (Listas)

En Python, los arreglos se conocen como listas y se utilizan para almacenar una colección de elementos ordenados. Aquí tienes un ejemplo de cómo declarar una lista y acceder a sus elementos:

      
        frutas = ['manzana', 'banana', 'cereza']
        print(frutas)  # Imprime toda la lista
        print(frutas[0])  # Accede al primer elemento de la lista
        print(frutas[1])  # Accede al segundo elemento de la lista

Operaciones con arreglos

Python proporciona diversas operaciones y métodos para trabajar con arreglos. Algunos ejemplos son:

  • len(lista): Retorna la longitud de la lista.
  • append(elemento): Agrega un elemento al final de la lista.
  • insert(2, 'nuevo'): Inserta un elemento en una posición específica de la lista.
  • remove('manzana'): Elimina la primera aparición del elemento especificado de la lista.
  • pop(1): Elimina y retorna el elemento en la posición especificada de la lista.

Aquí tienes un ejemplo que utiliza algunas de estas operaciones:

      
        frutas = ['manzana', 'banana', 'cereza']
        print(len(frutas))  # Imprime la longitud de la lista
        frutas.append('durazno')  # Agrega 'durazno' al final de la lista
        frutas.insert(2, 'naranja')  # Inserta 'naranja' en la posición 2
        print(frutas)  # Imprime la lista actualizada
        frutas.remove('banana')  # Elimina 'banana' de la lista
        elemento = frutas.pop(1)  # Elimina y retorna el elemento en la posición 1
        print(elemento)  # Imprime el elemento eliminado
        print(frutas)  # Imprime la lista actualizada
