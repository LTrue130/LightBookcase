Condicionales en Python

if-else

La estructura de control if-else permite ejecutar un bloque de código si se cumple una condición y otro bloque de código si no se cumple. Aquí tienes un ejemplo:

      
        edad = 20
        if edad < 18:
            print('Eres menor de edad')
        else:
            print('Eres mayor de edad')

if-elif-else

Si tienes múltiples condiciones, puedes utilizar la estructura if-elif-else para evaluarlas secuencialmente y ejecutar el bloque correspondiente al primer caso que se cumpla. Aquí tienes un ejemplo:

      
        nota = 85
        if nota >= 90:
            print('Excelente')
        elif nota >= 80:
            print('Muy bueno')
        elif nota >= 70:
            print('Bueno')
        else:
            print('Insuficiente')

Operadores de comparación

En los condicionales, puedes utilizar los siguientes operadores de comparación para evaluar las condiciones:

  • == (igual a)
  • != (diferente de)
  • < (menor que)
  • <= (menor o igual que)
  • > (mayor que)
  • >= (mayor o igual que)
