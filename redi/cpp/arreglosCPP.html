Arreglos en C++

Un arreglo en C++ es una estructura de datos que almacena una colección de elementos del mismo tipo. Cada elemento en el arreglo se identifica por un índice, que comienza desde 0. Aquí tienes un ejemplo de declaración de un arreglo:

      
        int arreglo[5]; // Declaración de un arreglo de tamaño 5

En este ejemplo, se declara un arreglo llamado "arreglo" que puede almacenar 5 elementos enteros. Los elementos del arreglo se acceden utilizando su índice, comenzando desde 0. Por ejemplo:

      
        arreglo[0] = 10; // Asigna el valor 10 al primer elemento del arreglo
        arreglo[1] = 20; // Asigna el valor 20 al segundo elemento del arreglo
        int valor = arreglo[2]; // Accede al valor del tercer elemento y lo guarda en una variable

Además de los arreglos unidimensionales, C++ también admite arreglos multidimensionales. Por ejemplo:

      
        int matriz[3][3]; // Declaración de una matriz 3x3
        matriz[0][0] = 1; // Asigna el valor 1 al elemento en la primera fila y primera columna
        matriz[1][2] = 5; // Asigna el valor 5 al elemento en la segunda fila y tercera columna
        int valor = matriz[2][1]; // Accede al valor del elemento en la tercera fila y segunda columna

Los arreglos en C++ son herramientas poderosas para almacenar y manipular conjuntos de datos. Son ampliamente utilizados en programación para diversas aplicaciones.
