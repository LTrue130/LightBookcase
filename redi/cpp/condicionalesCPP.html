Estructuras Condicionales en C++

La estructura if

La estructura if se utiliza para ejecutar un bloque de código si se cumple una condición. Aquí tienes un ejemplo:

      
        int edad = 18;

        if (edad >= 18) {
            std::cout << "Eres mayor de edad" << std::endl;
        }

La estructura if-else

La estructura if-else se utiliza para ejecutar un bloque de código si se cumple una condición y otro bloque de código si la condición no se cumple. Aquí tienes un ejemplo:

      
        int edad = 15;

        if (edad >= 18) {
            std::cout << "Eres mayor de edad" << std::endl;
        } else {
            std::cout << "Eres menor de edad" << std::endl;
        }

La estructura if-else if-else

La estructura if-else if-else se utiliza cuando hay múltiples condiciones y se ejecuta un bloque de código según la primera condición que se cumpla. Aquí tienes un ejemplo:

      
        int calificacion = 85;

        if (calificacion >= 90) {
            std::cout << "Excelente" << std::endl;
        } else if (calificacion >= 70) {
            std::cout << "Aprobado" << std::endl;
        } else {
            std::cout << "Reprobado" << std::endl;
        }

La estructura switch case

La estructura switch case se utiliza para evaluar una expresión y ejecutar diferentes bloques de código según el valor de la expresión. Aquí tienes un ejemplo:

      
        char operador = '+';
        int num1 = 10, num2 = 5, resultado;

        switch (operador) {
            case '+':
                resultado = num1 + num2;
                std::cout << "Suma: " << resultado << std::endl;
                break;
            case '-':
                resultado = num1 - num2;
                std::cout << "Resta: " << resultado << std::endl;
                break;
            case '*':
                resultado = num1 * num2;
                std::cout << "Multiplicación: " << resultado << std::endl;
                break;
            case '/':
                resultado = num1 / num2;
                std::cout << "División: " << resultado << std::endl;
                break;
            default:
                std::cout << "Operador inválido" << std::endl;
                break;
        }
