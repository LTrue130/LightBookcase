Bucles en C++

Que son

Los bucles en C++ son estructuras que permiten repetir un bloque de código varias veces. Existen tres tipos principales de bucles en C++:

1. Bucle for

El bucle for se utiliza cuando se conoce el número exacto de repeticiones. Aquí tienes un ejemplo:

      
        for (int i = 0; i < 5; i++) {
            // Código a repetir
        }

2. Bucle while

El bucle while se utiliza cuando se repite un bloque de código mientras se cumple una condición. Aquí tienes un ejemplo:

      
        int i = 0;
        while (i < 5) {
            // Código a repetir
            i++;
        }

3. Bucle do-while

El bucle do-while se utiliza cuando se desea ejecutar un bloque de código al menos una vez, y luego repetirlo mientras se cumple una condición. Aquí tienes un ejemplo:

      
        int i = 0;
        do {
            // Código a repetir
            i++;
        } while (i < 5);

Estos son los bucles principales en C++. Cada uno tiene su propio propósito y se utiliza en situaciones diferentes. Es importante comprender cómo funcionan y cuándo utilizar cada uno de ellos para escribir código eficiente y claro.
