Introducción a C++

¿Qué es C++?

C++ es un lenguaje de programación de propósito general que se utiliza ampliamente para el desarrollo de software. Es una extensión del lenguaje C e incorpora características adicionales, como programación orientada a objetos, que lo hacen más poderoso y versátil.

Sintaxis básica de C++

La sintaxis de C++ se compone de una serie de reglas que definen cómo se deben escribir los programas en este lenguaje. Aquí tienes un ejemplo de un programa simple en C++:

      
        #include <iostream>

        int main() {
            std::cout << "¡Hola, mundo!" << std::endl;
            return 0;
        }

Palabras clave en C++

C++ tiene un conjunto de palabras clave que tienen un significado especial y no pueden utilizarse como nombres de variables u otros identificadores. Algunas palabras clave en C++ incluyen:

  • int: define una variable entera.
  • float: define una variable de punto flotante.
  • if: inicia una estructura condicional.
  • for: inicia un bucle for.
  • class: define una clase.

Compilación de programas en C++

Para ejecutar un programa en C++, primero debes compilarlo. Puedes utilizar un compilador como g++ para generar un archivo ejecutable a partir del código fuente. Aquí tienes un ejemplo de cómo compilar un programa en C++ en la línea de comandos:

      
        // Compilación
        g++ programa.cpp -o programa

        // Ejecución
        ./programa
